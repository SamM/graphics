About Code Snippets

There are code snippets throughout this page that can be executed by clicking on them.

There are also variables that can be clicked to show their value.

Code snippets are executed in the following way:

var snippet = new Function("return 'Hello World';"); this.returned = snippet.call(window);

This means that to set a variable of the global scope the code snippets can use the this variable or window variable.

For example: if you click on this.returned and returned they will tell you that the function returned "Hello World"

Clicking on the following snippet will show an error message:

throw new Error("Woo hoo an error!");

Let's get started...

Creating an array

this.test = _("Hello", "World");
this.array = ["Hello", "World"]; this.test2 = _(array);

Getting the array

this.array1 = test(); this.array2 = test._;

array1 and array2 are the same variable.

array1 == array2 equals true

They both match array.

this.array3 = test2(); this.array4 = test2._;

array3 and array4 are both the same variable as array.

array == array3 && array == array4 equals true

Getting an item from an array

this.hello = test(0)(); this.world = test._[1];

hello equals "Hello".

world equals "World".

test(0)() returns the same as test._[0]

Both are ways to return a variable from an array.

this.world2 = test(-1)();

world2 equals "World".

world and world2 are the same variable.

Giving an index of -1 returns that last variable of the array.

Setting an item in an array

this.test = _("Hello", "World");
test(0)("Hi");

test() equals ["Hi","World"].

test._[1] = "Globe";

test() equals ["Hi","Globe"].

Setting variable from array to another value

this.numbers = _(1,2,3);

This makes a new array. Now we can make a new array of the same elements using the following code:

numbers(0,0,0,1,1,2, true);

The numbers 0,0,0,1,1,2 are all indexes to items in the original array.

numbers() now returns [1,1,1,2,2,3]

You can create a brand new array of numbers using the following code:

numbers(null)(3,4,5);

numbers() now returns [3,4,5]
